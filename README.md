# Shaders

Interactive WebGL shader experiments, built with Claude Code.

## figma-shader-orb

A self-contained shader design studio (`studio.html`) with four interactive
shaders — orb, lava, pixels, aura — each with live debug controls, per-shader
export (standalone HTML, preset JSON, 10s MP4), a live code editor, and a
hero-text overlay for landing-page previews.

**Try it:** download the repo, open `figma-shader-orb/studio.html` in any
modern browser. No build step, no dependencies — every file runs standalone.

See [figma-shader-orb/README.md](figma-shader-orb/README.md) for details.
