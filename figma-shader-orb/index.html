SHADER STUDIO

Eight interactive WebGL shaders for website backgrounds. Pick a tab, play with the canvas, tune the layers on the right — then export.

canvas
every shader reacts to your cursor — sweep, drag, stir. On web, drag nodes or empty space to rotate in 3D.
panel
sections are the shader's layers, in render order. Header checkboxes toggle a whole layer. Click any number to type an exact value; sliders nudge with ← → when focused.
export
the black export button opens the formats: html — a standalone file of the current shader for your site · preset — settings as JSON · png — current frame · mp4 — 3 / 5 / 10-second capture (interactions included).
saving
save settings keeps everything (all shaders, hero text, code edits) in this browser across reloads. Undo is unlimited: ⌘Z / ⇧⌘Z.
code
the bottom drawer shows the live engine source — edit, apply, copy. Broken edits revert themselves.
engine: orb
